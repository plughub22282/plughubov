\# VST3 Manager Project Context



\## Стек

Electron + Vite + React + Tailwind CSS + TypeScript.



\## Текущий статус

\- Настроена базовая архитектура Electron IPC.

\- Реализованы методы `stats`, `claim`, `redeem` для реферальной системы.

\- Разрабатывается 3-этапная система защиты от вирусов (Anti-Spoofing, двухфазный VirusTotal API v3, карантин-зона).



\## Архитектурные правила

\- Весь код строго разделен: Main (Node.js) и Renderer (React UI).

\- Взаимодействие только через `preload.ts` и `window.api`.



\## Design Context (impeccable)

\- Register: `brand` — весь десктоп-апп, включая утилитарные экраны (Key Manager, Settings), держит форму премиального минимала, а не «просто интерфейс».

\- Platform: `web` (Electron/Chromium).

\- North Star: «тихая студия ночью» — тёмная комната с одним источником света (Signal Violet), всё остальное намеренно в тени.

\- Позиционирование: безопасный кураторский доступ к VST3-экосистеме — редкость/курирование цепляет первой, безопасность закрывает решение.

\- Безопасность показывается визуально (статусы сканирования, карантин), не только текстом.

\- Жёсткий запрет: ни один экран не должен скатываться в шаблонный SaaS-дашборд или в пиратский сайт нулевых.

\- Полные документы: `PRODUCT.md` (стратегия), `DESIGN.md` (визуальная система, токены).
